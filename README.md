# Картотека и учёт обращений.
## Требования
- OpenServer (или любой PHP 7.4+ + MySQL)
- MySQL 5.7+/8.0+

## Установка
1) Склонируйте репозиторий и поместите проект в папку доменов OpenServer (например `OSPanel/domains/your-domain/`).

2) Создайте файл конфигурации:
- Скопируйте `config.example.php` → `config.php`
- В `config.php` укажите параметры подключения к БД и тестовые креды админа.

3) Создайте БД и таблицы:
- Создайте базу данных
- Выполните SQL-скрипт `schema.sql` для создания таблиц `authors`, `books`, `readers`

4) Запуск:
- Откройте сайт в браузере (через OpenServer домен)

## Использование
- **Главная страница:**  `index.php`
  - Поиск книг по названию или автору
  - В выдаче: название, автор, кол-во обращений читателей, сортировка по популярности
  - Каждый поисковый запрос автоматически добавляет обращение для каждой найденной книги

- **Админка:** `admin.php`
  - CRUD для книг и авторов
  - Логин/пароль задаются в `config.php` (`admin_user`, `admin_pass`)
